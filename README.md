## Case Studies

### REFAC Rwanda — National Education Platform  
https://refac.rw  

Built a national feedback system used to collect and manage education data across Rwanda.

- Designed scalable Django backend  
- Optimized performance for real-world usage (low bandwidth environments)  
- Delivered secure and structured data workflows  

Result: A stable platform supporting nationwide educational insights.

---

### Kwely — Hospitality SaaS Platform  
https://kwely.pro  

All-in-one system for restaurants, hotels, and service businesses.

- Booking system, orders, payments, dashboards  
- Real-time updates across admin and user interfaces  
- Multi-business support (SaaS model)  

Result: A production-ready platform with monetization potential.

### Golden Fashions — Business Platform

Custom website and system for a multi-sector business.

- Performance-focused deployment
